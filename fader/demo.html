Jquery Fader - innerfade with navigational control

Author: Philip Floetotto
Version: 1.0
Download: fader.zip (44 kb)

Introduction

jQuery.fader is a customization for the innerfade plugin created by http://medienfreunde.com/ I needed to have more navigational control and therefore added some extra bits. The plugin automatically creates links to jump to a specific slide and allows the user to browse the slides using "next" and "previous" buttons. You can stop the autocreation of the buttons and define your own navigational controls. (No example yet) Please have a look at the following examples to find out how it works.

Simple Fader with Images

The plugin sets the height of the faderContainer automatically by finding the first list element and adding the navigational height to it. An important feature was the "fading" of the navigational buttons. Instead of images, the links contain the number of the slide and can be styled using a css.

$('#imageList').fader({ 'speed': 600, 'timeout':3000 });

  • Testing

Simple Fader with more complex html

The fader takes any list element and turns it into a "slide". You can use any html like texts, images, links, etc.... Once the user interacts with the fader, the autoplay is turned off. By default the fader plugin navigation is set to "wrap", starting with the first one once the end is reached and vice versa.

$('#portfolio').fader({ 'speed': 600, 'timeout':3000 });

Simple Fader with animationType set to "slide"

You can easily add multiple faders to your page. Each fader is aware of its own slides, navigation controls and it's current state.

$('#portfolio1').fader({ 'speed': 600, 'timeout':3000, 'containerHeight':200, 'animationType':'slide' });
